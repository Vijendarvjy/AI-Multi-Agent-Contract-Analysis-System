# AI-Multi-Agent-Contract-Analysis-System
An AI-powered document analyzer app that automatically reads, extracts, and summarizes key information from files like PDFs, Word documents, and images. It uses NLP to identify insights, classify content, and improve productivity by reducing manual effort and speeding up decision-making.

---

# Key Technologies

- LangGraph
- LangChain
- OpenAI GPT
- Pinecone Vector Database
- Streamlit
- Python
